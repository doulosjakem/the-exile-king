## Formation Card Rules (Faction-Wide)

FORMATION CARDS

RULE:
- Formation cards represent specialized battlefield formations.
- When played, choose either the TOP or BOTTOM action.
- Only one formation effect from this card may be active.
- Place the card on the Command Board.
- Each formation has a limited number of ACTIVATIONS.
- Each time the formation triggers, spend 1 Activation.
- When the final Activation is spent, the card is immediately LOST.

---

## Deck Construction

- **Command deck** is built from army composition.
- **1 copy** per unit type brought, up to **2 copies** max per card type.
- **Starting hand:** 2 cards.
- **Draw per turn:** Up to hand size (hand fills to 4).
- **Fatigue:** Lose 1 random card after drawing each turn.
- **Cards chosen per turn:** 2.
- **Resolve:** Top of card A + Bottom of card B.

---

## Card Removal by Casualty

When the last unit of a type is eliminated, remove one matching command card from the deck.

| Unit Type Eliminated | Card Removed |
|---|---|
| Last David | David's Leadership |
| Last Refugee | Refugee Aid |
| Last Outcast | Outcast Tactics |
| Last Swordsman | Swordsmen Advance / Swordsmen Formation |
| Last Spearman | Spear Wall / Spearman Formation |
| Last Slinger | Slinger Skirmish / Slinger Formation |
| Last Archer | Archer Volley / Archer Formation |
| Last Scout | Scout Recon / Scout Formation |
| Last Veteran | Veteran Command |
| Last Mighty Man | Mighty Man Formation |

**Priority:** Remove from Deck first, then Hand, then Spent pile. Card moves to Lost pile permanently.
