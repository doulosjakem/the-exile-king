# Saul's Kingdom — Command Cards

> **Faction Philosophy:** Professional, disciplined army with heavy infantry and strong leadership. Command structure is rigid — units follow orders without the flexibility of David's deck system.

---

## Faction Overview

Saul's Kingdom army. Relationship varies by scenario — often pursuing David, sometimes fighting common enemies.

## Faction Rules

- Saul's Kingdom does **not** use the Command Card deck system. Instead, units follow a command token system based on initiative order.
- Abner (Commander) issues orders each turn: all units of the same type within 2 spaces of Abner receive his order.
- If Abner is eliminated, Saul's Kingdom loses 1 Activation from their command pool for the rest of the scenario.
- Elite Bodyguards protect Abner — they may spend an activation to intercept attacks targeting Abner.

- **Faction-wide formation cards:** See [formations/SAULS_KINGDOM.md](formations/SAULS_KINGDOM.md)

---

## Formation Card Rules (Faction-Wide)

FORMATION CARDS

RULE:
- Formation cards represent specialized battlefield formations.
- When played, choose either the TOP or BOTTOM action.
- Only one formation effect from this card may be active.
- Place the card on the Command Board.
- Each formation has a limited number of ACTIVATIONS.
- Each time the formation triggers, spend 1 Activation.
- When the final Activation is spent, the card is immediately LOST.

## Notes

- Saul's Kingdom is a professional army — heavier armor and better training than David's raw recruits.
- The rigid command structure (Abner as central authority) is a strength and a weakness: powerful when Abner is alive, brittle when he falls.
- Design note: Consider creating a simplified command card deck for Saul's Kingdom in future iterations to give them a similar mechanic to David's Company.
- For MVP, Abner's command token system is sufficient.
